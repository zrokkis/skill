# 5W1H 全维度背景优化技能 (5W1H Full Context Optimizer Skill)

本技能通过六个维度的强制性扫描，消除提示词中的“信息盲区”，确保 AI 能够像专家一样理解任务的每一个细微面，产出具备高度落地性的成果。

## 执行流程 (Optimization Workflow)

在需要构建复杂背景或撰写详尽计划时，请应用以下 5W1H 逻辑：

### 1. 人物 (Who) - 角色与受众
- **核心动作**：定义执行者（AI 的角色）和受益者（目标受众）。
- **操作指南**：明确“谁在做”和“做给谁看”。

### 2. 事件 (What) - 任务核心
- **核心动作**：精确描述要处理的实体或核心动作。
- **操作指南**：不要只写“写个报告”，要写“写一份关于 X 产品的竞品对比报告”。

### 3. 时间 (When) - 时效与跨度
- **核心动作**：设定任务的时间线或历史参考点。
- **操作指南**：包括截止日期、事件发生的频率或分析的时间窗口（如“过去 5 年”）。

### 4. 地点 (Where) - 空间与环境
- **核心动作**：设定事件发生的物理位置或虚拟平台。
- **操作指南**：是针对“美国市场”还是“公司内部网络”？

### 5. 原因 (Why) - 核心动机
- **核心动作**：解释任务的紧迫性和最终目的。
- **操作指南**：交代“为什么要现在做这件事”以及“成功后的期望”。

### 6. 方式 (How) - 路径与方法
- **核心动作**：规定使用的技术手段或风格。
- **操作指南**：指定具体的工具、数据来源、语气或特定的法律/技术规范。

## 优化示例 (Standard Template)

**初始需求**：“写个关于减少员工离职的计划。”

**5W1H 优化后**：
- **[Who]**：人力资源总监（执行者）面向公司董事会（受众）。
- **[What]**：制定一套全方位的“员工保留与关怀计划”。
- **[When]**：在接下来的两个季度内实施，参考过去一年的离职数据。
- **[Where]**：主要针对研发部门（离职率最高的部门）。
- **[Why]**：目前离职率已超过 20%，导致项目进度严重滞后并推高了招聘成本。
- **[How]**：通过优化薪酬结构、引入弹性工作制以及建立定期的 1 对 1 反馈机制来达成。
- **最终输出**：请根据上述要素撰写一份结构化、数据支撑的执行方案草案。
