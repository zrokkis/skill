# TAG 框架 (TAG Framework)

TAG 框架是一种精简且高效的提示词工程方法，专注于三个核心要素：任务、行动和目标。通过这种直观的结构，它能显著提升 AI 响应的针对性和结果导向性，特别适用于内容生成、问题解决、教学设计以及任何需要明确目标的任务场景。

## 核心组成部分 (Core Components)

1. **任务 (Task)**：清晰地识别提示词所针对的具体任务或挑战，为 AI 的参与设定舞台。
2. **行动 (Action)**：详细说明 AI 为响应提示词而预期采取的具体行动或一系列动作，引导其过程走向预期的成果。
3. **目标 (Goal)**：定义提示词的预期结果或目的，确保 AI 的响应与互动的最终目标保持一致。

---

## 提示词示例 (Prompt Examples)

### 示例 1：健康博客内容生成 (Health Blog Generation)
* **任务**：撰写一篇关于“水分补充对身体益处”的科普文章。
* **行动**：调研最新的关于水分补充的学术研究，并将研究发现合成为三个关键要点。
* **目标**：为读者提供关于如何保持水分补充的切实建议，从而促进其身心健康。

### 示例 2：软件 Bug 排查 (Software Bug Troubleshooting)
* **任务**：解决一个导致 Web 应用在移动设备上加载缓慢的问题。
* **行动**：分析提供的性能日志，识别资源加载的瓶颈，并列出 3 个优化 CSS 和 JS 加载顺序的方案。
* **目标**：将移动端首页的加载时间降低到 2 秒以内，提升用户留存率。
