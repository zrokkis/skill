# What If 框架 (What If Framework)

What If 框架是一个旨在激发想象力、探索替代方案、挑战假设并促进战略规划的思维工具。它通过“如果……会怎样？”的提问方式，引导用户和 AI 跳出当前的思维局限，探索各种假设情景，从而发现潜在的机遇、风险和创新解决方案。

## 核心组成部分 (Core Components)

1. **激发想象力 (Stimulate Imagination)**：提出“如果……会怎样？”的问题，打开通往无数可能性的大门，鼓励用户进行创造性思考，超越当前环境的限制。
2. **探索替代方案 (Explore Alternatives)**：审视不同的情景，帮助发现潜在的机会、风险和创新的解决方案。
3. **挑战假设 (Challenge Assumptions)**：通过质疑理所当然的事情，帮助识别可能阻碍进步的假设，并引入新的视角。
4. **促进战略规划 (Facilitate Strategic Planning)**：考虑各种“假设”情景对于战略规划至关重要，使组织能够为多样化的未来做好准备。

## 提示词示例 (Prompt Examples)

### 示例 1：自动驾驶的未来 (Future of Autonomous Vehicles)
* **任务**：探索城市交通的未来形态。
* **结构化提示词**：
    * **情景 (Context)**：想象一下正在探索城市交通的未来。
    * **假设提问 ('What if' Question)**：如果未来十年内，自动驾驶汽车成为城市地区的主要交通工具，会发生什么？
    * **影响 (Implications)**：这对城市规划、公共交通系统和碳排放会有什么影响？
    * **广泛影响 (Broader Impact)**：这种转变将如何影响城市生活方式、通勤模式和环境？

### 示例 2：教育变革 (Education Transformation)
* **任务**：构想未来的教育系统。
* **结构化提示词**：
    * **情景 (Context)**：传统的K-12教育模式正面临数字化冲击。
    * **假设提问 ('What if' Question)**：如果所有的学校都取消了固定的年级制，转而采用完全基于能力的个性化学习路径，会怎样？
    * **影响 (Implications)**：这将如何改变教师的角色、考试制度以及学生的社交互动？
    * **广泛影响 (Broader Impact)**：这种模式对缩小教育贫富差距有何潜在贡献或风险？
