# What If 假设推演技能 (What If Scenario Skill)

本技能通过强制开启“平行宇宙”视角，打破线性思维的桎梏。它不关注“现在是什么”，而是通过极端或颠覆性的假设，迫使 AI 推演“未来可能是什么”，从而发现被惯性思维掩盖的机遇与黑天鹅风险。

## 执行流程 (Optimization Workflow)

在需要进行前瞻性分析、创新突破或压力测试时，请应用以下 What If 逻辑：

### 1. 假设提问 ('What if' Question) - 设定奇点
- **核心动作**：抛出一个改变游戏规则的假设条件。
- **操作指南**：使用“如果 [核心变量] 发生了 [极端变化]...”，例如“如果公司失去了所有现有的销售渠道”、“如果重力只是现在的 50%”。

### 2. 影响推演 (Implications) - 连锁反应
- **核心动作**：推导第一轮和第二轮的直接后果。
- **操作指南**：询问对系统内部各要素的具体冲击，如“这对我们的现金流/组织结构/用户留存意味着什么？”。

### 3. 广泛影响 (Broader Impact) - 生态震荡
- **核心动作**：评估对外部环境或宏观层面的长远改变。
- **操作指南**：将视角拉高，询问对行业格局、社会伦理或人类行为模式的潜在重塑。

## 优化示例 (Standard Template)

**初始需求**：“帮我想想新产品的卖点。”

**What If 优化后**：
- **[Context]**：我们正在开发一款不依赖屏幕的新型智能手机。
- **[What if]**：如果未来的智能手机完全取消了视觉交互界面（屏幕），仅保留语音和触觉反馈，世界会怎样？
- **[Implications]**：
    1. 现有的 App 生态（抖音、Instagram）将如何崩溃或重构？
    2. 用户的隐私保护需求将发生什么质变（因为声音是广播的）？
- **[Broader Impact]**：这是否会导致人类重新从“低头族”变回“抬头族”？它对人类的注意力广度有何修复作用？
- **[Task]**：基于以上推演，为这款无屏手机提炼出 3 个最具颠覆性的“反直觉”卖点。
