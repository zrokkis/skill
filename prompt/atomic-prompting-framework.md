# Atomic Prompting 框架 (Atomic Prompting Framework)

Atomic Prompting 框架是一种专为 AI 图像生成设计的分层提示词工程方法。它借鉴了原子设计的概念，将提示词分解为从宏观到微观的四个层级：生物体 (Organism)、分子 (Molecule)、原子 (Atom) 和参数 (Parameters)。这种结构确保了对生成图像的精细控制，从整体构图到光照质感，实现极致的细节还原。

## 核心组成部分 (Core Components)

1. **生物体层级 (Organism Level)**：确立基础场景和艺术指导方向。这是图像的“骨架”，定义了主体是什么。
2. **分子层级 (Molecule Level)**：添加次级细节，进一步细化场景。这是图像的“肌肉”，增加了环境和互动的丰富性。
3. **原子层级 (Atomic Level)**：专注于复杂的细节，以增加深度和真实感。这是图像的“皮肤”，处理纹理、光影细微之处。
4. **参数层级 (Parameters Level)**：微调输出的技术方面，进行最后的润色。这是图像的“滤镜”，控制纵横比、风格化程度等技术指标。

## 提示词示例 (Prompt Examples)

### 示例 1：未来主义城市景观 (Futuristic Cityscape)
* **任务**：生成一张赛博朋克风格的城市图片。
* **结构化提示词**：
    * **生物体层级 (Organism Level)**：设想一个繁忙的、霓虹灯闪烁的未来城市。
    * **分子层级 (Molecule Level)**：突出摩天大楼与有机结构的融合，飞行汽车穿梭其中。
    * **原子层级 (Atomic Level)**：详细描绘被雨水打湿的街道反射出的霓虹灯光，以及路人身上机械义肢的金属质感。
    * **参数层级 (Parameters Level)**：追求具有动态光照的写实风格，纵横比 16:9，高分辨率。

### 示例 2：奇幻森林生物 (Fantasy Forest Creature)
* **任务**：创造一个从未见过的魔法生物。
* **结构化提示词**：
    * **生物体层级 (Organism Level)**：一只类似于鹿但拥有羽翼的神秘生物，栖息在发光的古老森林中。
    * **分子层级 (Molecule Level)**：它的角是由水晶构成的，周围环绕着飞舞的发光孢子。
    * **原子层级 (Atomic Level)**：羽毛上的微小鳞片纹理，以及水晶角内部折射出的彩虹光芒。
    * **参数层级 (Parameters Level)**：采用吉卜力工作室的画风，色彩饱和度高，柔光渲染。
