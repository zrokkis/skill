# 设想框架 (Imagine Framework)

设想框架通过引导 AI 进入“虚构”或“假设”的可视化模态，打破现实世界的约束。它以“想象...”作为启动词，邀请 AI 探索替代现实、未来发展或深层的同理心。这套框架特别适合头脑风暴、概念定义、科幻创作和政策影响评估。

## 核心组成部分 (Core Components)

1. **激发创意 (Inspire Creativity)**：以“想象...”开头，为探索和创造力设定舞台。这个词本身就是一种跳出平庸、开启潜能的邀请。
2. **假设场景 (Hypothesize Scenarios)**：使用提示词探索假设情境、潜在的未来发展或替代现实。核心是询问“如果...会怎样”并考虑其含义。
3. **展望方案 (Envision Solutions)**：鼓励制定在传统思维约束下可能无法出现的创新方案或构思。
4. **培养共情 (Cultivate Empathy)**：通过想象他人的经历、挑战或视角，增强对复杂人类问题的理解。
5. **探索后果 (Explore Consequences)**：深入探讨行动、决定或变化的潜在后果（社会、技术、环境或个人层面）。

## 提示词示例 (Prompt Examples)

### 示例 1：基因健康预测技术 (Genetic Health Prediction)
* **任务**：评估一项能够根据基因标志物预测健康风险的新技术。
* **结构化提示词**：
    * **激发创意 (Inspire Creativity)**：想象我们正在开发一种能够根据基因标志物准确预测个人健康风险的新技术。
    * **假设场景 (Hypothesize Scenarios)**：设想一个医疗资源根据这些预测进行自动分配的世界。
    * **展望方案 (Envision Solutions)**：这种创新将如何从全球范围内重塑医疗保健和患者护理的格局？
    * **培养共情 (Cultivate Empathy)**：考虑这其中的伦理含义，特别是对于那些被标记为高风险但无法承担治疗费用的个体。
    * **探索后果 (Explore Consequences)**：评估对社会保险制度、就业歧视以及人类对寿命预期心理的潜在影响。

### 示例 2：火星殖民生活 (Life on Mars)
* **任务**：设想在火星建立首个永久定居点。
* **结构化提示词**：
    * **激发创意 (Inspire Creativity)**：想象人类在火星上建立了第一个能够容纳 1000 人的自给自足的城市。
    * **培养共情 (Cultivate Empathy)**：想象一名在火星出生、从未见过地球天空的孩子的一天。
    * **探索后果 (Explore Consequences)**：探讨与地球断绝即时通信（20 分钟延迟）对社会治理和心理健康的影响。
