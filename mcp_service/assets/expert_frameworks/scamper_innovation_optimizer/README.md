# SCAMPER 框架 (SCAMPER Framework)

SCAMPER 框架提供了七种方法来探索和扩展创意。这是一种结构化的创新思维工具，能够帮助用户跳出固有思维模式，生成可操作的解决方案。

## 核心组成部分 (Core Components)

1. **替代 (Substitute)**：用另一个要素或流程替换现有的要素，以改善结果。
2. **合并 (Combine)**：将多个概念融合在一起，创造出新颖创新的东西。
3. **改造 (Adapt)**：修改想法以适应不同的需求、受众或环境。
4. **修改 (Modify)**：改变或放大某些元素，以增强价值或功能。
5. **改变用途 (Put to another use)**：为现有的想法或工具寻找新的应用场景或受众。
6. **消除 (Eliminate)**：移除不必要的部分，以简化流程并提高效率。
7. **反向 (Reverse)**：从相反的角度对想法进行重构，以获得独特的见解。

## 提示词示例 (Prompt Examples)

### 示例 1：产品开发 (Product Development)
* **背景**：改进一款可持续使用的水壶。
* **结构化提示词**：
    * **替代 (Substitute)**：我们可以用什么材料替换现有的塑料盖，使其更环保？
    * **合并 (Combine)**：我们能否将水壶与内置的过滤系统或温度显示器结合起来？
    * **改造 (Adapt)**：如何调整这款水壶，使其更适合户外攀岩者使用？
    * **修改 (Modify)**：如果我们将水壶的设计放大到 2 升，或者改变其抓握感，会发生什么？
    * **改变用途 (Put to another use)**：这款水壶的材料是否可以用于生产其他环保家居产品？
    * **消除 (Eliminate)**：包装中哪些部分是不必要的，可以被移除以减少浪费？
    * **反向 (Reverse)**：如果不考虑便携性，而是考虑稳定性，水壶的外观会有什么变化？

### 示例 2：数字营销 (Digital Marketing)
* **背景**：为一款健身应用策划营销活动。
* **结构化提示词**：
    * **替代 (Substitute)**：用短视频广告替换现有的静态图片广告。
    * **合并 (Combine)**：将健身挑战与社交媒体抽奖活动结合。
    * **改造 (Adapt)**：根据不同的节日（如春节）调整营销话术。
    * **消除 (Eliminate)**：移除注册流程中不必要的步骤，提高转化率。
