# RISE 系统化输出优化技能 (RISE Structured Output Skill)

本技能通过锁定执行链条的四个锚点（身份、素材、路径、标准），确保 AI 在处理复杂任务时不会偏离主线，产出具备极高可落地性的结构化成果。

## 执行流程 (Optimization Workflow)

在需要 AI 扮演特定身份并交付高质量成果时，请应用以下 RISE 逻辑：

### 1. 角色 (Role) - 身份定位
- **核心动作**：指定专家的职能和思考角度。
- **操作指南**：使用“你是一位具有 X 背景的 Y 专家...”。

### 2. 输入 (Input) - 资源约束
- **核心动作**：喂入原材料。
- **操作指南**：明确指出“基于以下附件/文本/链接资料...”、“考虑 [变量 A] 和 [变量 B]...”。

### 3. 步骤 (Steps) - 逻辑路径
- **核心动作**：拆解执行步骤。
- **操作指南**：列出分步任务，如“第一步清洗数据，第二步寻找异常，第三步解释原因”。

### 4. 预期 (Expectation) - 验收标准
- **核心动作**：定义最终交付物的形式和质量。
- **操作指南**：指定“以 JSON 格式输出”、“字数控制在 X 以内”、“必须包含 3 个具体案例”等。

## 优化示例 (Standard Template)

**初始需求**：“分析一下这篇文章。”

**RISE 优化后**：
- **[R] Role**：你是一位资深市场分析师。
- **[I] Input**：请基于下方粘贴的这篇关于“生成式 AI 行业现状”的深度调研文章。
- **[S] Steps**：
    1. 总结出文章中提到的 3 个主要市场机遇。
    2. 识别出潜在的 2 个行业风险。
    3. 找出支持这些断言的关键数据支撑。
- **[E] Expectation**：请输出一份 400 字以内的摘要，采用 Markdown 级联标题格式，语言风格要求客观、学术化。
