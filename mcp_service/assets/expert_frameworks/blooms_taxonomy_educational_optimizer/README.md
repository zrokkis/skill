# 布鲁姆分类提示框架 (Bloom's Taxonomy Prompt Framework)

布鲁姆分类框架通过六个分层级的认知维度，帮助用户构建能够激发 AI 不同深度思考能力的提示词。它广泛应用于教育、培训和深度学术研究，确保 AI 不仅仅是简单的“信息检索”，而是能进行更高阶的思维活动。

## 核心认知层级 (Core Cognitive Levels)

1. **记忆 (Remember)**：要求 AI 回收事实和基础信息。
2. **理解 (Understand)**：设计问题以探究对意义和概念的理解。
3. **应用 (Apply)**：制定要求在新情境中使用信息的任务。
4. **分析 (Analyze)**：开发剖析信息以理解其结构或背景的提示词。
5. **评估 (Evaluate)**：设计要求基于标准和准则做出判断的问题。
6. **创造 (Create)**：鼓励生成新的想法、产品或看待事物的新方式。

## 提示词示例 (Prompt Examples)

### 示例 1：编程学习路径 (Programming Learning Path)
* **任务**：学习 Python 装饰器。
* **分级提示词**：
    * **记忆 (Remember)**：Python 装饰器的基本定义是什么？
    * **理解 (Understand)**：请解释装饰器的工作原理，特别是它如何修改函数行为而不改变其源代码？
    * **应用 (Apply)**：编写一个简单的装饰器，用于计算任何函数的运行时间。
    * **分析 (Analyze)**：比较并分析使用装饰器与直接在函数内部添加逻辑的优缺点及对代码可维护性的影响。
    * **评估 (Evaluate)**：在大型生产环境中，过度使用装饰器是否是一个好的实践？请给出理由。
    * **创造 (Create)**：设计一个新的装饰器框架，允许根据用户的权限级别动态启用或禁用特定功能。

### 示例 2：历史研究案例 (Historical Research)
* **任务**：研究工业革命。
* **分级提示词**：
    * **分析 (Analyze)**：分析导致 18 世纪英国工业革命爆发的主要社会经济因素。
    * **评估 (Evaluate)**：根据当时劳动人口的生活质量标准，评估工业革命带来的长期收益是否超过了短期痛苦。
    * **创造 (Create)**：设想如果蒸汽机没有被发明，人类的工业进程可能会走向什么样的替代道路？撰写一个小场景。
