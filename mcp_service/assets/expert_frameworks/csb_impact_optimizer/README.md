# CSB 框架 (Challenge-Solution-Benefit Framework)

CSB 框架（挑战-方案-收益）是一种强大的叙事和演示结构。它专注于通过首先建立紧迫感（挑战），展示专业应对（方案），并最终聚焦价值产出（收益），来构建具有说服力的提示词。

## 核心组成部分 (Core Components)

1. **识别挑战 (Identify the Challenge)**：从清晰定义手头的问题或挑战开始。这为叙事设定了背景，并突出了对解决方案的需求。
2. **实施方案 (Implement the Solution)**：描述为应对挑战而采取的行动。这展示了解决方案背后的方法论和思维过程。
3. **突出收益 (Highlight the Benefit)**：以解决方案带来的积极结果或优势作为结尾。这展示了所采取行动的影响力和价值。

---

## 提示词示例 (Prompt Examples)

### 示例 1：软件开发复盘 (Software Development Post-mortem)
* **挑战 (Challenge)**：在双十一流量高峰期间，我们的支付网关出现了不可预测的间歇性失效。
* **方案 (Solution)**：我们紧急部署了熔断机制，并对下游服务进行了限流，同时增加了 30% 的服务器节点。
* **收益 (Benefit)**：成功在 15 分钟内恢复了系统，避免了约 500 万美元的潜在损失，并提高了系统的容错能力。
* **指令**：请根据上述 CSB 要素撰写一篇面向技术团队的内部事故复盘总结。

### 示例 2：营销案例包装 (Marketing Case Study)
* **挑战 (Challenge)**：某护肤品牌在年轻人群中的知名度不到 5%。
* **方案 (Solution)**：在抖音发起“ 21 天素颜挑战”话题，并邀请 50 位垂直领域的 KOL 参与互动。
* **收益 (Benefit)**：品牌话题阅读量突破 1 亿，官网转化率提升了 300%。
* **指令**：请以此结构起草一份向潜在广告主展示的成功案例推介词。
