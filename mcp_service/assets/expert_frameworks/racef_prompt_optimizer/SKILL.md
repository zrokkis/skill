# RACEF 提示词优化技能 (RACEF Prompt Optimization Skill)

本技能通过 RACEF 五步法，将通俗、模糊的初始指令转化为具备高职业素养和技术深度的“结构化提示词”。

## 执行流程 (Optimization Workflow)

当你需要优化一个提示词或任务时，请遵循以下 RACEF 逻辑：

### 1. 改写 (Rephrase) - 意图对齐
- **核心动作**：重新审视初始需求，将其翻译为 AI 更易理解的指令。
- **操作指南**：使用动作词（如“分析”、“评估”、“生成”、“解构”），消除歧义。

### 2. 追加 (Append) - 约束增强
- **核心动作**：补充技术细节、性能指标或排版要求。
- **操作指南**：
    - **格式约束**：Markdown 表格、JSON 结构、学术论文格式。
    - **风格约束**：专业客观、幽默风趣、极简主义。
    - **工具约束**：指定使用的库（如 Python 的 Pandas）或方法。

### 3. 情境化 (Contextualize) - 背景植入
- **核心动作**：定义 AI 的身份、任务的受众以及产出物的用途。
- **操作指南**：
    - **Role**：你是一位资深的资深系统架构师。
    - **Audience**：汇报对象是公司的决策层（C-level）。
    - **Purpose**：此报告将用于季度战略复盘。

### 4. 示例 (Examples) - 少样本引导 (Few-shot)
- **核心动作**：提供至少一个“输入-输出”的范例，锚定 AI 的生成轨迹。
- **操作指南**：给出期望的格式模板，特别是涉及到复杂逻辑或特定风格时。

### 5. 后续行动 (Follow-Up) - 闭环优化
- **核心动作**：预设反馈机制，允许 AI 进行自我修正或引导用户进一步补充。
- **操作指南**：添加诸如“在执行前，请先列出你的大纲供我确认”或“如果你需要更多背景，请在此询问”的指令。

## 优化示例 (Standard Template)

**初始需求**：“帮我写个市场调研报告。”

**RACEF 优化后**：
- **[R] 改写**：对 [特定领域] 进行深入的市场趋势挖掘与竞争格局分析。
- **[A] 追加**：涵盖过去 3 年的数据，包含 SWOT 分析及未来 5 年的增长预测。使用 Markdown 渲染。
- **[C] 情境化**：你是一位资深的战略咨询顾问，受众是风投机构的合伙人。语调需保持极度严谨。
- **[E] 示例**：参考 [范例链接/格式]，确保每一章都有核心洞察（Key Insights）。
- **[F] 后续**：请先输出报告的逻辑架构，待我确认后再进行全文填充。
