# 汉堡模型 (Hamburger Model)

汉堡模型是一种简单而有效的结构化写作框架。它通过将提示词内容比作汉堡的三个层次：上层引言（面包）、中间细节（肉层）和底层结论（面包），确保 AI 生成的内容具备清晰的逻辑、丰富的细节和完整的收尾。

## 核心组成部分 (Core Components)

1. **引言 (Introduction - 上层面包)**：从设定场景的提示词开始，提供背景信息或陈述待解决的问题。
2. **正文 (Body - 中间肉层)**：这是响应的核心部分，应深入探讨细节，通过示例、证据或分析来阐述主题。
3. **结论 (Conclusion - 下层面包)**：以引导 AI 总结关键点或提供解决方案的提示词结束，整齐地包裹住整个内容。

---

## 提示词示例 (Prompt Examples)

### 示例 1：可再生能源科普 (Renewable Energy Explained)
* **任务**：撰写一篇关于可再生能源的文章。
* **结构化提示词**：
    * **引言 (Introduction)**：介绍全球对可再生能源日益增长的需求背景，以及传统能源面临的危机。
    * **正文 (Body)**：详细叙述太阳能和风能的优势，包括对环境的影响、可持续性以及长期的经济效益，并辅以最新的行业数据。
    * **结论 (Conclusion)**：通过总结可再生能源如何有效应对全球能源挑战并展望未来的清洁能源图景来收尾。

### 示例 2：企业周报 (Business Weekly Report)
* **任务**：向领导汇报本周项目进展。
* **结构化提示词**：
    * **引言 (Introduction)**：简述本周项目的主攻目标和整体进度概况。
    * **正文 (Body)**：列出本周完成的 3 个关键任务，分析遇到的 2 个技术难点及其解决方案，并提供下周的工作预案。
    * **结论 (Conclusion)**：总结本周的执行效率评估，并提出一个关于资源分配的具体建议。
