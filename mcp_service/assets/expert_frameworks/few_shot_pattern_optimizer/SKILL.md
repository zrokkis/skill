# 少样本模式学习优化技能 (Few-Shot Pattern Optimizer Skill)

本技能利用 AI 的强大学习与模仿能力，通过“言传身教”（提供样本）而非仅仅是“文字描述”，确保输出结果的百分之百对齐。

## 执行流程 (Optimization Workflow)

在处理具有高度特定格式要求或复杂逻辑的任务时，请应用以下 Few-Shot 逻辑：

### 1. 指令 (Instruction) - 任务定调
- **核心动作**：用简洁的语言描述你要 AI 模仿的底层逻辑。
- **操作指南**：使用“请模仿以下模式进行处理”或“按照示例的格式提取信息”。

### 2. 演示/样本 (Demonstrations) - 黄金样本
- **核心动作**：提供 2-5 个经典的“输入-输出”对。
- **操作指南**：
    - 样本要具有代表性，涵盖不同的可能性。
    - 确保样本的格式（如 JSON、Markdown 表、甚至特殊的符号）与你最终想要的完全一致。

### 3. 查询 (Query) - 任务下发
- **核心动作**：给出你需要处理的真实数据。
- **操作指南**：紧接在最后一个样本之后，不要加多余的废话，让 AI 自然衔接。

## 优化示例 (Standard Template)

**初始需求**：“帮我把这些电影名字改成‘导演: 名字 - 电影: 名字’的格式。”

**Few-Shot 优化后**：
- **[Instruction]**：请根据提供的导演和作品信息，统一格式化为“导演: [姓名] - 电影: [片名]”。
- **[Demonstrations]**：
    1. 输入：克里斯托弗·诺兰，星际穿越
    2. 输出：导演: 克里斯托弗·诺兰 - 电影: 星际穿越
    3. 输入：宫崎骏，千与千寻
    4. 输出：导演: 宫崎骏 - 电影: 千与千寻
- **[Query]**：
    - 输入：斯皮尔伯格，辛德勒的名单
    - 输入：奉俊昊，寄生虫
