# GOPA 框架 (GOPA Framework)

GOPA 框架是一种面向目标的流程优化模型。它通过系统地定义目标、探索选项、权衡利弊并最终转化为行动，帮助用户和团队在复杂场景下保持决策的清晰度。这套框架非常适合战略规划、项目管理和需要精准执行方案的业务场景。

## 核心组成部分 (Core Components)

1. **目标 (Goals)**：清晰地定义目标和期望的结果，确保整个过程中的步调一致和焦点明确。
2. **选项 (Options)**：识别并列举所有可能的策略、路线或行动方案，以实现设定的目标。
3. **利弊 (Pros/Cons)**：评估每个选项的优缺点，分析其可行性和潜在影响。
4. **行动 (Actions)**：基于上述评估制定具体的行动计划，详细说明有效实施所选策略所需的步骤。

## 提示词示例 (Prompt Examples)

### 示例 1：业务市场扩张 (Market Expansion Strategy)
* **任务**：制定公司下一年的市场扩张计划。
* **结构化提示词**：
    * **目标 (Goals)**：在下一个财年内将市场份额提高 20%。
    * **选项 (Options)**：
        1. 引入新的产品线。
        2. 加大数字营销投入。
        3. 进入新的地理区域市场。
    * **利弊 (Pros/Cons)**：
        1. 新产品线：优点是使产品组合多样化；缺点是研发成本高。
        2. 数字营销：优点是具有成本效益，覆盖面广；缺点是需要专业人才。
        3. 新市场：优点是增长潜力大；缺点是需要深入的市场调研。
    * **行动 (Actions)**：
        1. 开展针对新产品的市场调研。
        2. 为数字营销分配专项预算。
        3. 确定首批扩张的目标区域市场。

### 示例 2：个人技能提升 (Personal Skill Development)
* **任务**：在三个月内掌握 Python 基础。
* **结构化提示词**：
    * **目标 (Goals)**：能够独立编写简单的自动化脚本。
    * **选项 (Options)**：参加在线课程、阅读专业书籍、通过项目进行实战。
    * **利弊 (Pros/Cons)**：课程有系统指导但费钱；书籍深入但反馈慢；项目实战快但基础可能不牢。
    * **行动 (Actions)**：每周完成 1 个小任务，每天学习 1 小时基础语法。
