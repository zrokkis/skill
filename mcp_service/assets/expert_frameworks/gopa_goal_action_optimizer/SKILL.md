# GOPA 目标执行优化技能 (GOPA Goal-Action Optimizer Skill)

本技能通过“方案赛马”与“风险对冲”，将模糊的目标转化为多维度的策略选择，并最终沉淀为可立即执行的任务清单。

## 执行流程 (Optimization Workflow)

在面临战略抉择或需要细化执行方案时，请应用以下 GOPA 逻辑：

### 1. 目标 (Goals) - 确定终点
- **核心动作**：定义可量化的成功标准。
- **操作指南**：使用关键结果（KR）进行描述，不要只写“想变好”，要写“提高 X 指标至 Y”。

### 2. 选项 (Options) - 发散思维
- **核心动作**：穷举至少 3 种达成目标的异构路径。
- **操作指南**：分别从成本领先、差异化创意、或是资源整合等不同维度给出备选方案。

### 3. 利弊 (Pros/Cons) - 压力测试
- **核心动作**：客观评估每个选项背后的隐形成本和收益。
- **操作指南**：考虑时间成本、资金消耗、人力门槛以及潜在的合规风险。

### 4. 行动 (Actions) - 落地生根
- **核心动作**：选取最优或组合方案，拆解为最小可执行单元。
- **操作指南**：以“下周动作”或“第一阶段里程碑”的形式输出。

## 优化示例 (Standard Template)

**初始需求**：“写个提高公司品牌知名度的办法。”

**GOPA 优化后**：
- **[G] Goals**：在三个月内，让目标行业内的品牌搜索指数（Brand Index）提升 15%。
- **[O] Options**：
    1. 举办线下行业高峰论坛。
    2. 与头部 KOL 合作进行内容植入。
    3. 在社交平台发起联名抽奖活动。
- **[P/C] Pros/Cons**：
    - 论坛：权威性高但周期长、成本贵。
    - KOL：见效快但受众精准度取决于选人，存在口碑风险。
    - 抽奖：扩散快但用户粘性差，可能吸来大量僵尸粉。
- **[A] Actions**：
    1. 筛选并签约 5 位垂直行业 KOL 进行深度评测。
    2. 同步在官微发起有奖互动，要求参与者必须分享真实使用心得。
    3. 监控每周关键词搜索增量。
