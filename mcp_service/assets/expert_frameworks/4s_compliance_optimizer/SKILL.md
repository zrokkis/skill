# 4S 合规与效率优化技能 (4S Compliance Optimizer Skill)

本技能通过引入“物理结构”与“伦理敏感性”，将提示词从单纯的命令提升为一种受控且具备社会责任感的交互。

## 执行流程 (Optimization Workflow)

在处理敏感任务或需要精简指令时，请持续应用以下 4S 逻辑：

### 1. 简单性 (Simplicity) - 降维打击
- **核心动作**：剔除废话，只留干货。
- **操作指南**：如果一句话能说完，就不要写一段。避免嵌套过深的从句。

### 2. 具体性 (Specificity) - 像素级定义
- **核心动作**：用数据和具体词汇取代模棱两可。
- **操作指南**：不写“写长一点”，而写“不少于 800 字”。不写“要有创意”，而写“使用隐喻手法”。

### 3. 敏感性 (Sensitivity) - 伦理雷达
- **核心动作**：预判输出的风险点。
- **操作指南**：考虑受众的性别、宗教、地域文化或其他潜在的情感触发点。要求 AI 保持中立或特定偏好时，需提供明确边界。

### 4. 结构性 (Structure) - 逻辑骨架
- **核心动作**：维持指令的叙事顺序。
- **操作指南**：遵循“前提 -> 任务 -> 细节 -> 输出”的黄金序列。

## 优化示例 (Standard Template)

**初始需求**：“帮我写个针对东南亚市场的促销文案，顺便别说错话。”

**4S 优化后**：
- **[Simplicity]**：文案需要包含 3 个卖点，旨在提高移动端下单量。
- **[Specificity]**：针对 Lazada 和 Shopee 平台，突出“免运费”和“首单 5 折”。
- **[Sensitivity]**：东南亚国家文化多样，避免使用任何可能引起宗教争议的符号（如特定的动物图案），语调需友好且尊重当地社交习俗。
- **[Structure]**：
    1. 社交媒体标题。
    2. 三点式核心详情。
    3. 引导关注与下单的 CTA。
