# ELI5 框架 (ELI5 Framework)

ELI5 代表“像我五岁一样解释它” (Explain Like I'm Five)。这套框架的核心思想是将极其复杂、技术性强或抽象的概念拆解为最基础、最通俗易懂的术语。它旨在消除专业壁垒，使非专业人士甚至儿童都能理解核心思想。

## 核心原则 (Core Principles)

1. **简单化 (Simplicity)**：只保留最核心的逻辑，剔除所有不必要的复杂性。
2. **去专业化 (No Jargon)**：严格禁止使用行业术语；如果必须使用，则需要用最简单的比喻进行解释。
3. **比喻化 (Use Analogies)**：使用日常生活中常见的场景（如玩积木、排队买冰淇淋）来类比抽象概念。
4. **受众导向 (Audience Focused)**：假设受众完全没有背景知识，思维方式天真且直接。

## 提示词示例 (Prompt Examples)

### 示例 1：解释“量子计算” (Explaining Quantum Computing)
* **任务**：向一个五岁小孩解释量子计算。
* **结构化提示词**：
    * **背景**：在科学课上，小朋友问起什么是超级计算机。
    * **指令**：请使用 ELI5 框架解释量子计算。
    * **要求**：不要使用“量子叠加”或“纠缠”这种词。用抛硬币或迷宫来做比喻。
    * **输出**：想象一个正常的计算机是一枚硬币，它要么是正面（0），要么是反面（1）。但量子计算机就像一个正在旋转的硬币，它在慢下来之前，既是正面也是反面，这让它能同时思考好多好多问题。

### 示例 2：解释“通货膨胀” (Explaining Inflation)
* **任务**：向门外汉解释为什么物价会涨。
* **结构化提示词**：
    * **指令**：请用 ELI5 的方式说明通货膨胀。
    * **要求**：用“神奇的森林”和“橡子”来做比喻。
    * **输出**：想象你住在森林里，大家用橡子换果子。以前 1 个橡子能换 1 个苹果。但如果森林里突然多出了超级多橡子，大家觉得橡子不稀奇了，以后就得用 2 个橡子才能换 1 个苹果了。这就是通货膨胀。
