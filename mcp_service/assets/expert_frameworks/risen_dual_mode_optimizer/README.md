# RISEN 框架 (RISEN Framework)

RISEN 框架是一个兼顾其精准性与灵活性的提示词工程模型。它通过组合角色 (Role)、输入 (Input/Instructions)、步骤 (Steps)、预期 (End Goal/Expectations) 和 狭窄/新颖性 (Narrowing/Novelty) 这五个要素，去除了传统框架的僵化结构。

注意：RISEN 的后两个字母 E 和 N 具有双重含义，这赋予了它极高的适应性：
*   **E (End Goal / Expectations)**：可以指最终的目标，或更广泛的期望。
*   **N (Narrowing / Novelty)**：可以指通过限制条件来“收窄”范围，也可以指引入“新颖性”来激发创新。

## 核心组成部分 (Core Components)

1. **角色 (Role)**：设定 AI 的角色，为预期的响应类型奠定基调。
2. **指令 (Instructions/Input)**：阐明你希望 AI 做什么，提供清晰的指令。
3. **步骤 (Steps)**：将任务分解为可管理的步骤，确保逻辑递进。
4. **预期 (End Goal / Expectations)**：定义提示词的最终目标，引导 AI 关注特定结果，或设定预期的成就标准。
5. **狭窄/新颖性 (Narrowing / Novelty)**：设定限制条件以收窄范围，或引入新颖元素以激发创造性思维（取决于任务需求）。

## 提示词示例 (Prompt Examples)

### 示例 1：市场营销活动策略 (Marketing Campaign Strategy)
* **任务**：为一款针对年轻专业人士的健身 App 制定营销活动。
* **结构化提示词**：
    * **角色 (Role)**：作为一名营销策略师。
    * **指令 (Instructions)**：为一款针对年轻专业人士的健身 App 开发一个为期 6 个月的营销活动。
    * **步骤 (Steps)**：包括目标、关键信息、目标平台和成功指标。
    * **预期 (End Goal)**：确保活动能引起目标人群的共鸣，并传达 App 的独特价值主张。
    * **新颖性 (Narrowing / Novelty)**：探索非传统的社交媒体趋势以最大化参与度。

### 示例 2：商业计划书 (Business Plan)
* **任务**：为一家环保时尚初创公司撰写商业计划。
* **结构化提示词**：
    * **角色 (Role)**：你是一位经验丰富的商业顾问。
    * **指令 (Instructions)**：撰写一份精简的商业计划书大纲。
    * **步骤 (Steps)**：涵盖执行摘要、市场分析、运营计划和财务预测。
    * **预期 (Expectations)**：文件应专业、数据驱动且具有说服力，适合向风险投资人展示。
    * **收窄 (Narrowing)**：重点关注欧洲市场的监管环境和消费者行为，忽略北美市场。
