# CRISPE 框架 (CRISPE Framework)

CRISPE 框架是一种专为 AI 提示词工程设计的方法论，强调**结构化精度 (Structured Precision)** 和 **迭代优化 (Iterative Refinement)**。它通过将清晰的任务定义与具体的约束条件和示例相结合，确保大语言模型（LLM）能够交付具有影响力、准确且可操作的输出。

## 核心组成部分 (Core Components)

1. **清晰度 (Clarity)**：清晰地定义任务或目标，消除歧义，确保 AI 精确理解目标。
2. **相关性 (Relevance)**：将提示词与特定的背景、受众或情境细节对齐，确保回应有的放矢。
3. **迭代 (Iteration)**：鼓励通过反馈和后续提示词进行优化，以改进 AI 的初始输出。
4. **针对性 (Specificity)**：提供精确的细节、数据或约束条件，缩小 AI 的关注范围，避免笼统的回应。
5. **参数 (Parameters)**：定义边界或限制（如长度、语气或格式），以有效地塑造输出结果。
6. **示例 (Examples)**：提供输入示例或预期的输出格式，以引导 AI 并演示预期效果。

## 提示词示例 (Prompt Examples)

### 示例 1：电商平台营销活动 (Marketing Campaign for an E-Commerce Platform)
* **清晰度 (Clarity)**：制定一项营销活动，以在季节性促销期间提升销售额。
* **相关性 (Relevance)**：重点针对 25-35 岁的年轻专业人士。
* **迭代 (Iteration)**：在各社交媒体平台上测试不同的广告格式。
* **针对性 (Specificity)**：包含一个有效期为两周的 8 折优惠券代码。
* **参数 (Parameters)**：输出应包含 3 个不同的广告文案版本，每个版本不超过 150 字。
* **示例 (Examples)**：参考服装行业类似的成功营销案例。

### 示例 2：员工培训计划 (Employee Training Program)
* **清晰度 (Clarity)**：创建一个培训计划，以提升员工的客户互动技巧。
* **相关性 (Relevance)**：重点关注如何处理高压力情境和客户投诉。
* **迭代 (Iteration)**：包含角色扮演脚本，以测试不同的应对方法。
* **针对性 (Specificity)**：针对电子产品领域的在线零售商量身定制内容。
* **示例 (Examples)**：参考领先电商公司已验证有效的培训方法。

### 示例 3：产品功能优先级排序 (Product Feature Prioritization)
* **清晰度 (Clarity)**：识别并排序与产品核心价值主张一致的功能。
* **相关性 (Relevance)**：重点关注能提升用户参与度和留存率的功能。
* **针对性 (Specificity)**：突出针对中小企业的功能。
* **参数 (Parameters)**：以表格形式输出优先级列表，包含“功能名称”、“影响度”和“开发难度”三列。
* **示例 (Examples)**：参考竞品的功能配置，以识别差距。
