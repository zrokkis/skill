# CO-STAR 冠军品质技能 (CO-STAR Champion Skill)

本技能源自新加坡首届 GPT-4 提示词大赛的冠军方案。它是在 **Context (背景)** 和 **Style (风格)** 两个维度上做得最极致的框架，能够将平庸的回答瞬间提升为大师级作品。

## 执行流程 (Optimization Workflow)

在希望获得“从 60 分直接提升到 95 分”的质变效果时，请应用以下 CO-STAR 逻辑：

### 1. 背景 (Context) - 提供语境
- **核心动作**：交代前因后果。
- **操作指南**：不要上来就下指令，先说“我是谁”、“我在做什么项目”以及“为什么要做这件事”。

### 2. 目标 (Objective) - 明确任务
- **核心动作**：定义直接产出。
- **操作指南**：“帮我写个代码”是弱目标；“编写一个 Python 脚本来自动化处理 Excel 数据”是强目标。

### 3. 风格 (Style) - 模仿大师
- **核心动作**：指定参考系。
- **操作指南**：这是本框架的灵魂。明确要求“像《经济学人》一样写作”、“使用鲁迅的笔触”或“模仿麦肯锡的咨询报告风格”。

### 4. 语气 (Tone) - 情感调优
- **核心动作**：设定情绪温度。
- **操作指南**：“严肃”、“幽默”、“激昂”或“冷峻”。

### 5. 受众 (Audience) - 对象感
- **核心动作**：定义阅读者。
- **操作指南**：“给 5 岁的孩子看”和“给 50 岁的博士生导师看”，输出内容会有天壤之别。

### 6. 响应 (Response) - 格式规范
- **核心动作**：规定物理形态。
- **操作指南**：JSON、表格、Markdown、Python 代码块或 CSV。

## 优化示例 (Standard Template)

**初始需求**：“写个关于远程工作的帖子。”

**CO-STAR 优化后**：
- **[C] Context**：随着疫情结束，很多公司强制要求员工回办公室，引发了员工的不满。我正在运营一个职场吐槽号。
- **[O] Objective**：撰写一篇引发强烈共鸣的社交媒体帖子，批评“一刀切”的强制返岗政策。
- **[S] Style**：犀利、讽刺，类似于脱口秀演员的段子风格。
- **[T] Tone**：幽默中带着无奈，甚至有点愤世嫉俗。
- **[A] Audience**：25-35 岁、深受通勤之苦的城市白领。
- **[R] Response**：一段 200 字以内的短文，包含 3 个相关的 Emoji，结尾带一个互动问题。
