# RICE 框架 (RICE Framework)

RICE 框架是一种不仅限于优先级的评估方法，它是评估想法潜在价值和可行性的战略指南。通过四个核心要素，RICE 提供了一种定量与定性相结合的决策方式，帮助你有效地分配资源，实现影响力的最大化。通过为每个想法计算简单的 RICE 分数，你可以优先处理那些投资回报率 (ROI) 最高的任务。

## 核心组成部分 (Core Components)

1. **覆盖范围 (Reach)**：衡量有多少人或系统将从这项改进中受益。
2. **影响力 (Impact)**：评估这项改动将带来的潜在价值或提升。
3. **信心 (Confidence)**：根据数据或以往经验，评估你对该想法成功的把握程度。
4. **努力程度 (Effort)**：估算实施该改动所需的资源（时间、金钱和人力）。

## 提示词示例 (Prompt Examples)

### 示例 1：SaaS 产品功能优先级排序 (Feature Prioritization for SaaS Product)
* **角色 (Role)**：作为一名产品经理。
* **任务 (Task)**：为一款 SaaS 产品提出功能优先排序策略。
* **结构化提示词**：
    * **覆盖范围 (Reach)**：衡量有多少用户或系统将从该功能中受益。
    * **影响力 (Impact)**：评估该功能将带来的潜在价值。
    * **信心 (Confidence)**：评估你对该功能想法成功的确定程度。
    * **努力程度 (Effort)**：估算实施该功能所需的资源（时间、人力）。

### 示例 2：营销活动策划 (Marketing Campaign Planning)
* **角色 (Role)**：作为一名营销策略师。
* **任务 (Task)**：为一款新型环保产品设计营销活动。
* **结构化提示词**：
    * **覆盖范围 (Reach)**：估计该活动想法将覆盖或惠及多少人。
    * **影响力 (Impact)**：评估该活动对产品成功的潜在价值提升。
    * **信心 (Confidence)**：根据数据或经验评估活动成功的确定性。
    * **努力程度 (Effort)**：估算执行该活动所需的资源和人力。
