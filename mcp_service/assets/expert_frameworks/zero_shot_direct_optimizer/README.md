# 零样本提示技术 (Zero-Shot Prompting Technique)

Zero-Shot（零样本）提示词技术是一种不向模型提供任何支持示例或演示，仅通过任务描述来引导 AI 执行任务的方法。其目的是评估并利用模型在预训练过程中积累的内在知识，它是所有高级提示词工程方法的基础。

## 核心概念 (Core Concepts)

1. **定义 (Definition)**：在提示过程中仅提供任务描述，不附带任何示例。
2. **目标 (Objective)**：利用模型的预训练知识库，通过清晰的指令直接获得结果。
3. **应用 (Application)**：它是建立任务基准测试的核心手段，适用于模型已经非常熟悉的通用型任务。

## 提示词示例 (Prompt Examples)

### 示例 1：情感分析 (Sentiment Analysis)
* **指令**：请判断以下评论的情感是积极的、消极的还是中立的。
* **输入**：这顿饭的味道真是出乎意料的好。
* **输出**：[由 AI 直接生成]

### 示例 2：事实查询 (Fact Retrieval)
* **指令**：请列出意大利最著名的三个历史遗迹，并简要说明其历史意义。
* **输入**：无（直接下达指令）
* **输出**：[由 AI 直接生成]
