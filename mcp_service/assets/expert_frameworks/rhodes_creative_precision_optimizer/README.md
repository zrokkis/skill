# RHODES 框架 (RHODES Framework)

RHODES 框架是一个将创造力与精确性相结合的提示词构建策略。它通过引入“感知检查 (Sense Check)”这一独特环节，确保 AI 在生成内容之前完全理解用户的意图。该框架非常适合创意写作、市场营销以及任何对风格和语调有极高要求的场景。

## 核心组成部分 (Core Components)

1. **角色 (Role)**：定义 AI 的角色，以设定响应的基调和视角。
2. **目标 (Objective)**：清晰地阐述提示词的目标，聚焦 AI 的努力方向。
3. **细节 (Details)**：提供明确的细节和参数，以指导 AI 的响应。
4. **示例 (Examples)**：使用示例来说明期望的风格、语调或输出格式。
5. **感知检查 (Sense Check)**：确认 AI 对提示词的理解，确保在执行前达成一致。

## 提示词示例 (Prompt Examples)

### 示例 1：科幻小说创作 (Sci-Fi Storytelling)
* **任务**：创作一个未来主义的短篇故事。
* **结构化提示词**：
    * **角色 (Role)**：作为一名极具想象力的科幻作家。
    * **目标 (Objective)**：写一个引人入胜的故事，背景设定在一个科技与自然完美融合的未来城市。
    * **细节 (Details)**：故事应探讨先进科技与环境可持续性之间的和谐共存。
    * **示例 (Examples)**：想象一下《银翼杀手》遇上《阿凡达》，将高科技的城市景观与郁郁葱葱的绿色避难所结合起来。
    * **感知检查 (Sense Check)**：你理解我想要的创意方向和主题元素吗？（请先确认理解再开始写作）

### 示例 2：品牌重塑提案 (Rebranding Proposal)
* **任务**：为一家传统茶叶公司撰写品牌重塑方案。
* **结构化提示词**：
    * **角色 (Role)**：你是一位擅长传统品牌年轻化的资深品牌策略师。
    * **目标 (Objective)**：制定一份旨在吸引 Z 世代消费者的品牌更新提案。
    * **细节 (Details)**：保留品牌的百年历史底蕴，但视觉和语言风格要通过“国潮”和“极简主义”进行现代化改造。
    * **示例 (Examples)**：参考故宫文创的转型案例，将古老符号转化为现代流行ICON。
    * **感知检查 (Sense Check)**：在生成正式提案前，请先告诉我你打算从哪三个维度切入？
