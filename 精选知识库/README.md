# 精选知识库 (Curated Knowledge Base)

## 1. 核心定义 (Core Definition)
本目录用于收录** Erin **亲自摘抄、整理并沉淀的高阶 Prompt 框架、业务 Skill 文档及核心方法论。这些内容构成了 AI 协作的底层能力池。

## 2. 目录结构 (Directory Structure)
*   `精选SKILL/`: 存放按特定框架优化的原子级 Skill 文档。
*   `方法论/`: 存放通用的逻辑协议与操作规程（如 L.A.F.R. 协议）。

## 3. 维护准则 (Maintenance Guidelines)
1.  **原子化 (Atomicity)**：保持文档粒度适中，确保每一个具体业务 ID 和对照表可全局搜索。
2.  **双语锚定 (Bilingual Anchoring)**：专业术语首次出现时需标注英文原词。
3.  **学术化风格 (Academic Tone)**：保持客观、中立，拒绝冗余。
4.  **无损合并 (Zero-Loss Principle)**：在进行任何文档重构时，严禁精简或丢弃原始内容细节。
